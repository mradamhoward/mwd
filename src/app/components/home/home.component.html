Our Engineering Process

Communication stage is where we converse with our clients to write up the requirements. Feasability assessments will be made to ensure that it is possible to produce the product.

Planning stage is where we will turn the requirements into chronological steps to develop some type of architecture for the system.

Modeling is when we turn the steps produced in planning into mockup drawings. Photoshop and Pencil are used to create wireframes.

Development is when we turn the wire frames into first draft product/system. Using an iterative approach we develop the draft into further design types to produce the best quality result.

Deployment is when we are happy with the development stages produce and put the application live. This may be publishing to AWS or the respective mobile app stores.

What is Responsive Web Design?

Responsive Web Design is the process of making a website that scales and adapts to the device it is running on. When it is on a phone it will scale to a portrait mode which looks better.
